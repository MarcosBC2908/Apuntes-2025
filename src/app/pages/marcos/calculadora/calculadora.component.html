  1. Exlicar ngModel y como se importa para utilizarlo

    2. ngModel es una directiva en Angular que se utiliza para crear un enlace bidireccional entre un modelo de datos en el componente y un campo de formulario en la vista. Esto significa que cualquier cambio en el campo de formulario se refleja automáticamente en el modelo de datos y viceversa. La parte "model" en ngModel se refiere a este modelo de datos que se está utilizando en la aplicación

    Para utilizar la directiva ngModel en un proyecto de Angular, se necesita importar el módulo correspondiente en un archivo de configuración principal, que generalmente es app.module.ts. Para hacerlo se debe:

    1. Importar FormsModule: Abrir el archivo app.module.ts en un proyecto Angular y añadir la importación de FormsModule desde angular/forms. Esto es esencial para que ngModel funcione correctamente.

    2. Uso de ngModel: Una vez que se haya importado FormsModule, puedes utilizar ngModel en los componentes.
